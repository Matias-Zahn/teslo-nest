# Teslo API

1. Clonar este repositorio.
2. `npm install`
3. Clonar el archivo `env.template` y renovabralo a `.env`
4. Cambiar los varoles del `.env`

5. Levantar la base de datos

```
docker-compose up
```

6. Ejecutar el Seed

```
http://localhost:3000/api/seed
```

7. Levantar el pryecto:

```
npm run dev
```
